run via ex2.py file (if tested in the 'planet.cs.biu.ac.il' server skip to section 4b):

1. open Linux Shell or cmd (Command Shell) in the location dir of the file: ex2.py

2. make sure python is insatlled (latest 3. version preffered):
	a. in cmd, type: python --version
	b. in Linux Shell, type: python3 --version

3. make sure packages are insatlled by typing:
	pip install numpy matplotlib

4. run the ex2.py file and add the input 'test.txt' file for example:
	a. for cmd, type:
			python ex2.py test.txt
	b. for Linux Shell, type:
			python3 ex2.py test.txt
